# 📊 电子产品电商销售与用户行为分析 (E-commerce Sales & User Behavior Analysis)

## 📝 项目背景
基于一份包含 **56万+** 条记录的电子产品销售数据集，利用 Python 进行全方位的数据挖掘。项目不仅分析了销售趋势和产品分布，更深入探究了用户的生命周期价值。通过构建用户行为标签和 **K-Means 聚类算法**，对用户进行了精细化分层，为精准营销和用户留存提供了数据支持。

## 🛠️ 技术栈
* **语言:** Python 3.x
* **数据处理:** Pandas, NumPy
* **可视化:** Matplotlib, Seaborn
* **机器学习:** Scikit-learn (K-Means Clustering)
* **分析方法:** 漏斗分析, 复购/回购率计算, RFM模型变种, 聚类分析

## 💡 核心发现与工作内容

### 1. 数据清洗与预处理
* 处理缺失值与重复值，转换时间序列数据。
* 提取时间特征（年、月、日），并处理异常价格数据（剔除0元订单）。

### 2. 销售趋势分析
* **月度增长率 (MoM):** 识别出 5月-7月 为销售高峰期（可能受年中大促影响），11月数据存在缺失或异常下降。
* **品牌分析:** Samsung 销量占据主导地位，Apple 客单价较高。男性在高价位产品上的消费能力略高于女性。
* **地域分布:** 用户主要集中在北上广等一线经济发达地区。

### 3. 用户生命周期与行为分析
* **复购率与回购率:** 计算了每月的用户复购率（约 35.56%）和回购率，监控用户留存健康度。
* **用户状态分层:** 将用户划分为 `New` (新客), `Active` (活跃), `Unactive` (不活跃), `Return` (回流) 四类，绘制了用户状态流转图。

### 4. 用户价值分层 (K-Means 聚类)
利用机器学习算法将用户划分为三个核心群体：
* **Cluster 0 (低价值/流失用户):** 占比最大，活跃度低。
* **Cluster 1 (中等价值/发展用户):** 消费频率适中，具有培养潜力。
* **Cluster 2 (高价值/核心用户):** 活跃度高，复购意愿强，是重点维护对象。
